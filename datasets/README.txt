Consult your YOLO version documentation for correct placement of files. Below is the typical folder structure for YOLO versions up to 4.

/user/home/darknet        - <<location of executable>>
/user/home/darknet/cfg    - <<location of configuration files>>
/user/home/darknet/data   - <<location of data files>>
/user/home/darknet/backup - <<location of weight files>>

Edit MBS.data to do more configuration if needed (e.g. set paths to test and validation files).

If using YOLOv3, move the annotation text files from `data/images` to `data/labels`.
